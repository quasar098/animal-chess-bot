# openings

opening lines for animal chess
can be mirrored (g1e3 == a1c3)

```
Start position
├── mongolian opening (g1e3)
│   └── warehouse defense (b7c5)
└── mount opening (f1e3)
    ├── duel response (b7c5)
    └── cicada gambit (g7f5)
        ├── accepted (e3f5)
        └── declined (e3c4)
```
